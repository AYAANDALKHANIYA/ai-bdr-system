# My Project

Hi Grandma,

I built a system that helps companies send smart emails to potential customers automatically.

Instead of a human writing emails one by one, my system:
1. Takes a list of people (from a CSV file)
2. Understands their company and industry
3. Thinks about what that company might need
4. Writes a personalized email for them
5. Gives a score on how good that lead is

So basically, it acts like a **digital sales assistant**.

---

# What makes it “AI”?

The system uses Artificial Intelligence (AI) to:
- Understand companies
- Generate insights
- Write human-like emails

It doesn’t copy text — it creates new content every time.

---

# How it works (in simple terms)

1. User uploads a file with names and companies
2. My backend reads the file
3. AI processes each row
4. It generates:
   - Insights
   - Email
   - Score
5. Results are shown on the website
